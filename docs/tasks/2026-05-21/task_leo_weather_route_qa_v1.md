## 二、路线决策台校对

### 主入口

只校对三类主入口：

- 徒步

### 三类字段差异

请分别校对字段。

徒步必须关注：

- 分段
- 水源
- 补给

- 天气窗口

骑行 / 摩旅必须关注：

- 路面类型
- 车流强度
- 补给间距
- 维修点

- 长下坡
- 隧道
- 落石
- 大车风险
- 骑行和摩旅的区别提示

登山必须关注：

- 大本营
- 高营地
- 冲顶窗口
- 折返点
- 技术等级
- 必备装备
- 高反风险
- 天气红线
- 冰雪 / 冰裂缝 / 落石风险
- 救援可达性
